# CPU Benchmark Tool

一个漂亮的移动端CPU跑分工具，支持单核和多核性能测试。

## 功能特性

- ✅ 单核性能测试
- ✅ 多核性能测试（使用Web Workers）
- ✅ 12项基准测试
- ✅ 实时进度显示
- ✅ 动画效果
- ✅ 响应式设计

## 在线访问

部署在 GitHub Pages：[访问链接]

## 本地运行

```bash
# 启动本地服务器
python3 -m http.server 8080

# 访问
open http://localhost:8080
```

## 测试项目

1. File Compression - 文件压缩
2. Navigation - 导航路径计算
3. HTML5 Browser - HTML5浏览器
4. PDF Renderer - PDF渲染
5. Photo Library - 图片处理
6. Clang - 代码编译
7. Text Processing - 文本处理
8. Asset Compression - 资源压缩
9. Object Detection - 对象检测
10. Background Blur - 背景模糊
11. Horizon Detection - 水平线检测
12. Object Remover - 对象移除

## 技术栈

- HTML5
- CSS3 (Modern Design)
- JavaScript (ES6+)
- Web Workers (多核测试)
